# Terminal Hangman with Persistent Leaderboard

A classic Hangman game built in Python that features a dynamic scoring system and a persistent leaderboard to track player performance across sessions.

##  Features
* **Dynamic Word Loading:** Words are imported from a separate module to keep the core logic clean.
* **Persistent Leaderboard:** Player scores are saved to a local text file and updated automatically.
* **Duplicate Guess Protection:** Prevents players from losing lives by guessing the same letter twice.
